## Caracteristicas principales de typescript

### 1. Tipos de datos
    - any  (cualquier tipo de dato)
    - array (arreglo)
    - enum (enumeracion)
    - void (no devuelve nada)
    - null (null)
    - undefined (undefined)
    - never (no se puede)
    - object (objeto)
    - typeof (tipo de dato)
    - symbol (simbolo)
    - bigint (numeros grandes)
